# CLCOMX 0.1.2

릴리즈 성격: 패치 릴리즈

## 변경점

- 탭별 보조 터미널 dock을 추가했습니다.
  - 현재 탭의 `distro + workDir` 기준 일반 셸 실행
  - 숨김 후 다시 열면 같은 셸 유지
  - 셸이 직접 종료된 경우 다음 열기 때 새 셸 생성
- 보조 터미널을 메인 에이전트 터미널을 밀지 않는 overlay dock 구조로 바꿔, 열고 닫을 때 메인 xterm reflow가 덜 발생하도록 정리했습니다.
- 보조 터미널 포커스 경로를 보강해, 입력 중 `Space`가 토글 단축키로 오인식되던 문제를 수정했습니다.
- 터미널 설정에 보조 터미널 토글 단축키와 기본 높이 옵션을 추가했습니다.
- 터미널 링크/파일 열기, 장세션 복구, 보조 터미널 흐름을 현재 UI 기준으로 문서화했습니다.
- `terminal-aux` E2E 팩에 보조 터미널 재사용과 `Space` 입력 회귀 검증을 추가했습니다.

## 검증
