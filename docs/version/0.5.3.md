# CLCOMX 0.5.3

릴리즈 성격: 패치 릴리즈

## 변경점

- 앱 셸과 세션 탭 흐름의 도메인 경계를 정리했습니다.
  - 탭 닫기, 이름 변경, pin/lock, 순서 이동, 창 간 이동, 새 창 분리 흐름을 전용 controller로 분리했습니다.
  - 앱 시작, 창 종료, overlay visibility, preview action handoff를 app-shell 영역으로 정리했습니다.
- 터미널과 에디터 통합 구조를 분리했습니다.
  - 에디터 hydration, mode transition, path open, terminal-editor preflight를 개별 controller로 나눴습니다.
  - embedded editor, quick open, close confirm, assist/aux/draft panel 조립을 terminal feature view로 이동했습니다.
- 런처와 탭 UI의 feature 경계를 강화했습니다.
  - launcher view/controller/contracts를 launcher feature 아래로 정리했습니다.
  - tab context menu, activation/focus bridge, drag reorder, TabBar view 위치를 session-tabs feature 중심으로 정리했습니다.
- 터미널 loading lifecycle과 aux/draft UI 조립을 정리해 유지보수성과 회귀 테스트 범위를 개선했습니다.
