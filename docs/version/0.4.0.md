# CLCOMX 0.4.0

릴리즈 성격: 마이너 릴리즈

## 변경점

- Claude 세션 시작 옵션에 `--enable-auto-mode`를 기본 적용했습니다.
  - 새 세션을 시작할 때 Claude가 auto mode 순환을 노출할 수 있도록 시작 플래그를 붙입니다.
  - 세션 복구 시에도 같은 플래그를 유지해 새 세션과 복구 세션의 동작을 맞췄습니다.
- 터미널 설정에 Claude 전용 startup flag 옵션을 추가했습니다.
  - `Claude Code` 블록에서 auto mode 노출 여부를 켜고 끌 수 있습니다.
  - 구조를 `claudeCliFlags`로 묶어 이후 Claude startup flag가 더 생겨도 같은 경로에 확장할 수 있습니다.
- Claude 명령 조립 경로를 공통화했습니다.
  - start/resume 명령이 같은 인자 빌더를 사용합니다.
  - 기존 Codex 및 기타 에이전트의 기본 동작은 유지됩니다.

## 검증

- `npm run test`
- `npm run check:frontend`
- `cargo test --manifest-path src-tauri/Cargo.toml`
- `bash scripts/build-dev.sh`

## 비고

- `--enable-auto-mode`는 Claude Code 공식 permission modes 문서를 기준으로 확인했습니다.
- 실제 auto mode 사용 가능 여부는 Claude 플랜과 모델 조건에 따라 달라질 수 있습니다.
