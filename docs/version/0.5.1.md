# CLCOMX 0.5.1

릴리즈 성격: 패치 릴리즈

## 변경점

- 드래프트 입력 UI를 리사이즈 가능한 오버레이 패널로 전환했습니다.
  - 메인 터미널 레이아웃을 밀지 않고 보조 터미널과 같은 오버레이 계층으로 표시됩니다.
  - 보조 터미널과 동시에 열리지 않으며, 한쪽을 열면 다른 쪽은 자동으로 닫힙니다.
- 드래프트 높이 동작을 정리했습니다.
  - 처음 열 때는 1줄 높이로 시작합니다.
  - 사용자가 조절한 높이는 다시 열어도 유지됩니다.
  - 다시 조절할 때는 마지막 높이에서 시작하되, 원래 최소 높이까지 줄일 수 있습니다.
- 브라우저 프리뷰와 실제 앱의 동작을 더 가깝게 맞췄습니다.
  - 드래프트/보조 터미널 토글과 높이 유지 동작을 프리뷰에도 동일하게 반영했습니다.
  - 프리뷰 회귀 테스트를 추가해 패널 전환과 높이 유지 흐름을 검증합니다.
- 터미널 설정을 단순화했습니다.
  - 드래프트 최대 줄 수 옵션은 제거하고, 드래프트 오버레이 동작 안내만 남겼습니다.
