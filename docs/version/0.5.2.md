# CLCOMX 0.5.2

릴리즈 성격: 패치 릴리즈

## 변경점

- 터미널 파일 링크 해석을 보강했습니다.
  - 세션별 PTY runtime snapshot에 저장된 실제 셸 HOME을 우선 사용해 `~/...` 경로를 더 안정적으로 해석합니다.
  - bare filename 후보 탐색과 세션 컨텍스트 기반 해석 계약을 정리해, 프런트와 백엔드가 같은 세션 기준으로 경로를 해석하도록 맞췄습니다.
- 홈 디렉터리 symlink 경로 처리를 수정했습니다.
  - `~/.claude/skills/...`처럼 symlink를 경유하는 경로는 먼저 WSL 안에서 실제 경로로 canonicalize한 뒤 Windows 경로로 변환합니다.
  - 이 변경으로 `.claude/skills` 같은 링크형 디렉터리 아래 파일도 에디터 열기 대상에 포함됩니다.
- 관련 테스트를 추가했습니다.
  - 세션 기반 HOME 우선순위
  - 세션 컨텍스트 mismatch fallback
  - symlink 경유 홈 경로 canonicalization
