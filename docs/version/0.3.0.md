# CLCOMX 0.3.0

릴리즈 성격: 마이너 릴리즈

## 변경점

- 최근 히스토리 관리를 보강했습니다.
  - Recent 목록에서 히스토리 항목을 개별 삭제할 수 있습니다.
  - 삭제 전 확인 모달을 표시합니다.
  - 삭제 결과를 `tab_history.json`에 즉시 반영합니다.
- 터미널 렌더러 선택을 추가했습니다.
  - 설정에서 DOM과 WebGL 렌더러를 선택할 수 있습니다.
  - WebGL은 opt-in이며 일부 환경에서 처리량이 더 좋을 수 있습니다.
- 창 이동 시 라이브 터미널 복원을 보강했습니다.
  - 런타임 중 탭을 다른 창으로 옮길 때 현재 화면 상태를 더 정확히 복원합니다.
  - redraw가 많은 Claude Code, Codex 같은 TUI에서 attach 품질을 높였습니다.
- 터미널 attach와 출력 처리를 안정화했습니다.
  - UTF-8 decoding을 보강했습니다.
  - 기존 PTY 재연결 시 출력 replay 품질을 높였습니다.
  - 장시간 세션 복원 시 깨진 화면이나 중간 출력 누락 가능성을 줄였습니다.
- 터미널 UX를 다듬었습니다.
  - 메인 터미널과 보조 터미널의 loading overlay를 보강했습니다.
  - 단축키 입력 처리와 Ctrl+C 인터럽트 흐름을 정리했습니다.
  - `+` 키를 포함한 단축키 인식 안정성을 높였습니다.
- Claude Code 장세션 화면 안정성을 보강했습니다.
  - footer redraw로 인한 ghosting 완화 실험을 추가했습니다.
  - soft follow와 Unicode 폭 처리 관련 기반을 정리했습니다.

## 검증

- `npm run test`
- `npm run check:frontend`
- `cargo test --manifest-path src-tauri/Cargo.toml`
- `bash scripts/build-dev.sh`

## 비고

- 이번 단계는 버전 반영과 저장소 반영까지를 포함합니다.
- 태그 생성과 배포 릴리즈는 별도 지시가 있을 때만 수행합니다.
