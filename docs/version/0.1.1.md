# CLCOMX 0.1.1

릴리즈 성격: 패치 릴리즈

## 변경점

- `claude`, `codex` 멀티 에이전트 세션 모델을 정리하고, 에이전트 선택/복원 흐름을 안정화했습니다.
- 종료 시 resume token을 캡처해 `workspace.json`, `tab_history.json`에 저장하고, 워크스페이스 복원과 최근 히스토리 재오픈 시 같은 세션으로 이어지도록 보강했습니다.
- 새 세션 런처를 재설계해 에이전트/배포판 선택, 디렉토리 탐색, 내부 스크롤, 상단 툴바 구조를 정리했습니다.
- 터미널에 URL 링크 메뉴를 추가하고, 파일 경로 클릭 시 Windows 에디터로 여는 기능을 추가했습니다.
- 에디터 감지에 VS Code, Cursor, Windsurf, PhpStorm, Notepad++, Sublime Text를 포함했습니다.
- 장세션 복구 시 스크롤이 위로 튀는 현상을 줄이기 위해 복구 직후 tail-follow 동작을 보강했습니다.
- E2E 테스트 팩을 현재 UI와 기능에 맞게 갱신하고, `terminal-links`, 장출력 `workspace-restore` 검증을 추가했습니다.
- README, 테스트 문서, 인수인계 문서를 현재 구조 기준으로 갱신했습니다.

## 검증

- `npm run check`
- Windows Tauri E2E 기능팩 수동 검증
