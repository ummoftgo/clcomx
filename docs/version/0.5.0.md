# CLCOMX 0.5.0

릴리즈 성격: 마이너 릴리즈

## 변경점

- 브라우저 프리뷰 모드를 추가했습니다.
  - Tauri 앱을 실행하지 않고도 앱 셸을 브라우저에서 열어 프런트 디자인을 확인할 수 있습니다.
  - `preview.html` 진입점과 preview runtime mock을 통해 탭, 설정, 런처, 드래프트/보조 패널까지 같은 셸로 볼 수 있습니다.
- 프리뷰와 실제 앱의 UI 공유 범위를 넓혔습니다.
  - 터미널 하단 assist/draft 패널과 보조 터미널 프레임을 공용 컴포넌트로 분리했습니다.
  - 프리뷰 전용 터미널은 mock 본문만 담당하고, 앱 셸과 주요 패널 UI는 실제 앱 코드와 같은 구조를 사용합니다.
- 런타임 스타일 레이어와 사용자 CSS 오버라이드 경로를 정리했습니다.
  - 런타임 테마 적용을 별도 스타일 레이어로 옮겨 구조를 단순화했습니다.
  - 상태 루트의 `custom.css`를 읽어 마지막 레이어에서 덮어쓸 수 있게 해 프런트 조정 경로를 마련했습니다.
- 브라우저 프리뷰 상태 전환을 보강했습니다.
  - `workspace / dense / empty` 프리셋과 프레임 폭 전환을 추가했습니다.
  - 프리뷰 툴 패널은 접고 다시 열 수 있으며, 접힌 상태에서도 상단 액션을 가리지 않도록 여백을 조정했습니다.
  - 프리뷰에 노출되는 경로 표기는 실제 작업 디렉토리 대신 `/home/user/work/project` placeholder로 고정했습니다.

## 검증

- `npm run test`
- `npx tsc --noEmit`
- `npm run build`
- `cargo test --manifest-path src-tauri/Cargo.toml`
- `bash scripts/build-dev.sh`
