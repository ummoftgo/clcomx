# CLCOMX 0.5.4

릴리즈 성격: 패치 릴리즈

## 변경점

- 앱 셸과 터미널 화면의 view/controller 경계를 추가로 정리했습니다.
  - 앱 dialog stack을 별도 view로 분리했습니다.
  - 터미널 overlay, loading, runtime surface 조립을 feature view로 이동했습니다.
  - 터미널 renderer, paint, DOM helper, aux resize, shortcut routing 로직을 전용 controller/helper로 분리했습니다.
- Preview runtime을 더 작은 도메인 모듈로 분리했습니다.
  - editor file cache, history command, preset data, bridge command 처리를 개별 모듈로 나눴습니다.
  - Preview runtime은 facade 역할에 집중하도록 축소했습니다.
- Editor navigation의 언어별 resolver 경계를 정리했습니다.
  - JS/TS/Svelte import parser를 독립 모듈로 분리했습니다.
  - PHP `use`/class path resolver를 독립 모듈로 분리했습니다.
  - alias import/use, Svelte script block, PHP class path 우선순위에 대한 회귀 테스트를 보강했습니다.
- 로컬 작업용 계획 파일이 릴리즈 소스에 섞이지 않도록 `.agent-works/`를 ignore 처리했습니다.
